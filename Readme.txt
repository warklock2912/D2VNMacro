Cài python trước khi chạy
https://www.python.org/downloads/
Chạy file install để cài đặt
Chạy file start để khởi động Macro.
Tạo room trước khi chạy. Kiến nghị sử dụng cài đặt -m 125.

	F3 -> F6 - Các skill buff

3. Mở máu monster lên. (không nhớ là auto skill dùng click bừa hay theo boss nữa :D )

4. Tele to waypoint
	Mỗi act thì dùng toạ độ trên máy khác nhau. Cách ngu đang dùng là cái Mouse_Control coi toạ độ. Có cách hay hơn thì dùng

5. Tele to key 1. 
	Cùng tương tự tele to waypoint
